HOW TO USE

This application is designed to provide you Open Issues for Software AG repositories.

You can start searching with a topic name or repository name(s)

Steps:

  1. Select how you want to search (Repository/Topic name)
  2. Type the search field as shown in the search bar and click 'search icon' or press ENTER
  3. Please wait patiently
  4. Once the data is loaded, you can filter data by:
    • Repository name
    • Issue Number
    • Title
    • Body
    • Created/Updated Date Time
  5. Click on switch to get Pull Requests OR Issues
  6. Hide columns you don't wish to see:
    • click on 'Hide Columns' dropdown
    • select/deselect fields
  7. You can SORT the data with respect to Repository name:
    1. Hover over Repository header
    2. Click on the Arrow button
