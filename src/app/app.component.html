Repository Issues

How to use

This application is designed to provide you Open Issues for Software AG repositories.

You can start searching with a topic name or repository name(S)

Steps:

  1. Select how you want to search (Repository/Topic name)
  2. Type the search field as show in the search bar and click 'search icon' or press ENTER
  3. Please wait patiently
  4. Once the data is loaded, you can filter data by:
    • Repository name
    • Issue Number
    • Title
    • Body
    • Created/Updated Date Time
  5. You can SORT the data with respect to Repository name
    • Hover over Repository header
    • Click on the Arrow button
